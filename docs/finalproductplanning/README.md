# yunmao 最终规划文档入口

`docs/finalproductplanning/` 是 yunmao 项目当前的**权威规划入口**。早期调研材料保留在 `docs/earlyproductplanning/` 仅作历史背景；本目录文档与早期文档冲突时，以本目录为准。

## 3. 与早期目录的关系

- `docs/earlyproductplanning/` 内容来源于 feed-the.cat 报道、早期方案探索、AI 写出的多份草案。**保留为历史背景**，可作为创意灵感、市场数据、硬件供应链信息的参考。
- 凡是写入本目录的结论与原则，与早期文档冲突时一律以本目录为准。
- 早期文档涉及的某些玩法（大胃王榜、激光笔、虚拟工会等）已在 10 章明确标注“不采纳/暂缓”。

## 4. 使用约定

- 产品范围变化：先改 00、03、05、10；架构/技术选型变化：先改 01、02、04、06、08、09；决策变化：补 07 的 ADR。
- 指标定义变化：先改 11，再同步到 06、10、05。
- 当前权威路线：Phase 1 起 Web/H5 与 iOS/Android App 同步交付核心体验；小程序、TV、桌面端、Open SDK 在未来 Phase 视情况推进。
- 文档要可执行：每个阶段都要有交付物、验收标准、风险、监控和回滚方案。
- 文档变更走 PR；架构级变更需要架构组评审（见 07 第 6 节、08 第 8.3 节）。
